# Pricing Calculator

Небольшое приложение на Streamlit для расчета инструментов в биномиальной модели процентной ставки.

Приложение строит:

- дерево краткосрочных ставок;
- матрицу цен бескупонной облигации `ZCB_10`;
- матрицу цен `ZCB_t`;
- цену форварда на бескупонную облигацию;
- цену фьючерса;
- цены американского call-опциона на фьючерс для двух страйков.


## Что внутри

- `app.py` - интерфейс Streamlit и вывод результатов.
- `engine.py` - построение матриц и все основные формулы.
- `models.py` - входные параметры, служебные структуры и результаты расчета.
- `requirements.txt` - зависимости проекта.

## Используемая модель

В приложении используются параметры:

- `T` - общий срок;
- `n` - число шагов;
- `r0` - начальная краткосрочная ставка;
- `sigma` - волатильность;
- `t` - срок для `ZCB_t` и форварда;
- `k` - срок экспирации фьючерса и опциона;
- `E1`, `E2` - страйки опционов.

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

Если используется локальное окружение из проекта:

```bash
.\.calc_env\Scripts\streamlit.exe run app.py
```
